# Sai Baba Vedha Garden

A modern, divine-themed website for Sai Baba Vedha Garden built with Next.js 14, React 18, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **UI Library:** React 18
- **Language:** TypeScript 5
- **Styling:** Tailwind CSS
- **UI Components:** Radix UI
- **Icons:** Lucide React
- **Utils:** clsx, tailwind-merge, class-variance-authority

## Features

- 🏠 **Home Page** - Beautiful hero section with divine theme
- 📅 **Events** - View upcoming events and celebrations
- 💝 **Donation** - Support the temple with donations
- 🕉️ **Pooja Services** - Book special poojas and ceremonies

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

