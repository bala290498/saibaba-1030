# Audio Folder

Place your MP3 audio file here for background music.

## Required File

- `background-music.mp3` - The audio file that will play automatically throughout the website

## Usage

The audio player component will automatically load and play the file at `/audio/background-music.mp3`.

## Notes

- Supported formats: MP3, OGG, WAV
- Recommended: MP3 format for best browser compatibility
- File size: Keep it reasonable for faster loading (under 5MB recommended)
- The audio will loop automatically
- Users can control play/pause and volume
